# Create Pixel Chibi Bag System - Production Version

## Overview

Create a new production-ready pixel art and chibi themed bag system in a separate folder with modular file structure. The system will have a retro pixel art aesthetic with cute chibi vibes, vibrant colors, and pixelated UI elements while maintaining all existing functionality.

## File Structure

```
pixel-chibi-bag/
├── index.html              # Demo HTML file
├── styles/
│   └── pixel-chibi.css    # Pixel chibi theme styles
├── scripts/
│   ├── bag-system.js      # Core bag system logic
│   └── game-data.js       # Game data (skills, items, gear, etc.)
└── README.md              # Documentation
```

## Theme Design - Pixel Chibi

- **Color Palette**: 
  - Bright pastels: #ff6b9d (pink), #c44569 (rose), #f8b500 (yellow), #4ecdc4 (cyan), #95e1d3 (mint)
  - Retro game colors: #2d3436 (dark), #636e72 (gray), #00b894 (green), #0984e3 (blue
