# Changelog

All notable changes to the Transfer module will be documented here.

## 1.0.0
- Initial public API: init, addPlayer, addNPC, addHouse, updateEntity, remove, get, list
- Interiors overlay with manifest-driven content
- Manifest schema and lightweight validator
- Pluggable projection via setProjection(fn)
- Interaction router: hitbox sizing, hover/focus classes
- rAF-batched position updates
- Diagnostics panel and structured logging
- Theming via CSS variables and setTheme('dark'|'light')
- Accessibility: ARIA roles, keyboard activation
- i18n: setLocale(url) and t(key) for labels
- Base path in manifest and path resolver
- Samples and test harness
