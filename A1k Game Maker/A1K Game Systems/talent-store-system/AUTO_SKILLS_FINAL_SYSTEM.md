## üéÆ How To Use

1. Open `standalone.html` in any browser
2. Choose an auto-skill lane (SHADOW, SPIRIT, LIGHTNING, VOID, PHOENIX, or MONARCH)

5. Click "Equip" button in talent tree
6. Check "My Talents" tab to see active bonuses
7. Repeat for other lanes!
8. Unlock Dual Wielding to equip 2 skills simultaneously
