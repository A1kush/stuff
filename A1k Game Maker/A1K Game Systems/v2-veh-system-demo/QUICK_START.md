## Troubleshooting

**Vehicle won't move?**
- Make sure you pressed E to board it
- Check you're pressing arrow keys

**Can't see "Press E" prompt?**
- Walk closer to the vehicle
- Interaction radius is about 60 pixels

**Character not visible?**
- They appear when riding the vehicle
- Position varies by vehicle type

**Particles not showing?**
- They appear when vehicle moves
- Different types for different vehicles

**Want to change character/style mid-game?**
- Press ESC to return to gallery
- Change settings
- Select vehicle again
