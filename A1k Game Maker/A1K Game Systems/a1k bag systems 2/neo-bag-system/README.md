## Usage

### Offline Use
1. Open `index.html` in a browser
2. The bag system will auto-open
3. Press 'B' to toggle, 'ESC' to close

### With HTTP Server (Recommended)
```bash
cd neo-bag-system
python3 -m http.server 8080
```
Then open: `http://localhost:8080/index.html`
